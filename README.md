# Paisabazaar Credit Risk in Banking

## Project Type
Exploratory Data Analysis (EDA)

---

## Project Overview

This project focuses on analyzing credit risk in the banking sector using a Paisabazaar dataset.  
The objective is to identify patterns and insights related to loan approvals, applicant profiles, and financial behavior.

Through exploratory data analysis, the project uncovers relationships between customer attributes and credit risk, helping financial institutions better understand lending patterns.

---

## Problem Statement

Banks and financial institutions face challenges in identifying high-risk loan applicants.  
Analyzing customer data can help detect patterns that indicate potential credit risk.

This project analyzes customer financial data to identify key factors influencing loan approval and creditworthiness.

---

## Dataset Information

The dataset includes information about applicants such as:

- Applicant income
- Loan amount
- Credit history
- Property area
- Education
- Self-employment status
- Loan approval status

## Key Insights

Some of the important insights from the analysis include:

- Credit history plays a major role in loan approval.
- Applicants with higher income have a higher chance of loan approval.
- Loan approval rates vary across property areas.
- Self-employed applicants show different loan patterns.
